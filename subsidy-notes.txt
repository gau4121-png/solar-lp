== 蓄電池・太陽光 補助金情報 ==

■ 国の補助金：DR補助金（DRに対応したリソース導入拡大支援事業）
- 2025年実績: 予算66.8億円、7月2日に予算満了（約2ヶ月で終了）
- 2026年: 予算58億円、1ヶ月足らずで予算満了
- 上限額: 最大60万円
- 計算方法: 蓄電池商品工事代の1/3 or 初期実効容量1kWhあたり3.7万円の低い方
- 条件: 蓄電容量1kWhあたり13.5万円以下（販売目標価格）
- DR条件: 2027年3月31日まで需給ひっ迫時に遠隔制御される可能性

■ EP CUBE（カナディアンソーラー）のDR補助金額
- EP CUBE 9.9kWh → 初期実効容量9.4kWh → 補助金394,800円
- EP CUBE 13.3kWh → 初期実効容量12.6kWh → 補助金529,200円

■ 長州産業のDR補助金額
- SPVマルチ 9.8kWh → 初期実効容量8.3kWh → 補助金365,200円
- SPVマルチ 12.7kWh → 初期実効容量10.9kWh → 補助金479,600円
- SPVマルチ 16.4kWh → 初期実効容量14.1kWh → 補助金549,900円

■ 埼玉県の補助金（家庭における省エネ・再エネ活用設備導入補助金）
- 太陽光発電: 7万円/kW（上限35万円）
- 蓄電池: 10万円（一律）
- セット導入で最大45万円
- 注意: 太陽光は令和7年6月26日で受付終了、蓄電池は引き続き申請可能
- 条件: 既存住宅、FIT認定不可、発電量30%以上自家消費
- 国の補助金と併用不可

■ 朝霞市の補助金
- 太陽光発電: 3.5万円/kW（上限10万円）
- 蓄電池: 10万円（一律）
- 申請期間: 2025/4/21〜2026/2/27
- 注意: 令和7年9月4日で受付終了済み
- 既存住宅のみ対象、事前申請制、窓口提出のみ

■ 新座市の補助金
- 太陽光発電: 9万円/kW（上限45万円）
- 蓄電池: 上限45万円
- 申請期間: 要確認

■ 東京都の補助金（災害にも強く健康にも資する断熱・太陽光住宅普及拡大事業）
- 蓄電池: 12万円/kWh（上限なし）
- DR実証参加で+10万円
- 例: 9.9kWhのEP CUBEなら 12万×9.9 = 約118.8万円 + DR10万 = 約128.8万円
- 例: 13.3kWhのEP CUBEなら 12万×13.3 = 約159.6万円 + DR10万 = 約169.6万円
- 東京都は全国で最も手厚い補助金

■ 補助金併用のまとめ
- 埼玉県の場合: 埼玉県補助金 + 市区町村補助金（併用可）、ただし国のDR補助金とは併用不可
- 東京都の場合: 東京都補助金 + 国のDR補助金 + 市区町村補助金（併用可）
- 東京都は非常にお得
